Fragmentation threshold

Fragmentation allows to split an IP packet in a burst of smaller fragments transmitted on the medium. In most cases this adds overhead, but in a very noisy environment this reduces the error penalty and allow packets to get through interference bursts. This parameter sets the maximum fragment size; a value equal to the maximum packet size disable the mechanism.

Examples:

  • iwconfig eth0 frag 512
  • iwconfig eth0 frag off
